# 🧠 高效能 K 線圖表引擎 - 操作指南

這是一個從零開始實作、不依賴任何第三方圖表庫的「高效能 K 線圖表引擎」。採用 Canvas 2D 分層渲染技術，專為處理大量資料與流暢交互設計。

---

## 📁 專案結構 (Project Structure)

```bash
/src
  /core
    DataManager.ts        # 資料中心：負責 Lazy Load、快取與 Mock 資料生成
    ViewportEngine.ts     # 視窗引擎：控制顯示範圍 (Index)、縮放與捲動邏輯
    ScaleEngine.ts        # 比例引擎：負責價格與像素座標的精確轉換 (Price <-> Y)
    RenderEngine.ts       # 渲染引擎：Canvas 繪製核心，支援分層渲染 (Grid/Candle/Overlay)
    InteractionEngine.ts  # 交互引擎：滑鼠拖曳、滾輪縮放、慣性滑動與十字線
    IndicatorEngine.ts    # 指標引擎：計算 MA、EMA 等技術指標（支援增量更新）
    LoaderController.ts   # 載入控制器：監控邊界觸發歷史資料載入
  /types
    Candle.ts             # 基礎資料型別定義
  /utils
    math.ts               # 數學工具：線性映射、數值鉗制、格式化
  main.ts                 # 程式入口：初始化引擎並啟動渲染循環
index.html                # 分層畫布配置與樣式
package.json              # 環境配置 (Vite + TypeScript)
tsconfig.json             # TypeScript 編譯設定
```

## 🧩 核心模組說明

### 1. 渲染優化 (Canvas Layering)
- **Grid Canvas**: 繪製背景網格，僅在顯示範圍變化時更新。
- **Candle Canvas**: 繪製 K 線實體與影線，使用 `requestAnimationFrame` 確保 60 FPS。
- **Overlay Canvas**: 繪製十字線與資訊標籤，滑鼠移動時僅更新此層，避免重繪整張圖表。

### 2. 縮放與捲動 (Zoom & Scroll)
- **Anchor Zoom**: 縮放以滑鼠位置為中心，確保畫面不飄移。
- **Inertia Scroll**: 支援滑鼠拖拽後的慣性滑行效果，提升交互手感。

### 3. 資料管理 (Lazy Loading)
- 初始載入 1000 筆。
- 向左滑動至邊界時，自動觸發 `loadMoreHistory`。
- 載入後自動補償 `startIndex`，確保畫面不會跳動。

## 🚀 啟動與開發

### 1. 安裝環境
確保您的電腦已安裝 [Node.js](https://nodejs.org/)。

### 2. 安裝依賴
在專案根目錄執行：
```bash
npm install
```

### 3. 啟動開發伺服器
```bash
npm run dev
```
啟動後，開啟瀏覽器訪問 `http://localhost:5173` 即可看到圖表。

## 🧪 測試與功能驗證

### 1. 效能測試
- 預設生成 1000 筆資料，您可以嘗試快速捲動與縮放。
- 即使資料量擴展至 10 萬筆，由於 `RenderEngine` 僅渲染可見範圍 (Visible Range)，效能依然穩定。

### 2. Lazy Load 驗證
- 將圖表向右拖拽（即查看左側歷史資料）。
- 當滑動到最左側時，上方會顯示「Loading History...」，隨後歷史資料會無縫拼接。

### 3. 精確度驗證
- 移動滑鼠觀察十字線，確認十字線與 K 線中心點精確對齊。
- 縮放時觀察特定 K 線，確認其不會隨著縮放而橫向漂移。

## 💡 技術亮點
- **無依賴開發**: 純 TypeScript + Canvas 2D。
- **動態 LOD**: 根據 `candleWidth` 自動調整繪製細節。
- **極低內存佔用**: 僅維護必要的資料陣列，渲染不產生多餘 DOM 節點。
